# SenDon - Premium Healthtech Platform
## Product Requirements Document

**Created:** December 2025
**Version:** 1.0
**Status:** Frontend MVP Complete

---

## 📋 Original Problem Statement

Create a premium, modern, and emotionally powerful bilingual (French/English) website for SenDon - a healthtech startup that connects blood donors, hospitals, and CNTS (National Blood Transfusion System) using AI-powered predictive intelligence.

The website must feel world-class (Apple-level design, Stripe-quality UI) with smooth animations, refined typography, and a strong emotional narrative centered on saving lives.

---

## 🎯 Core Requirements

### Design Standards
- Minimalist, elegant, modern aesthetic
- Colors: white, deep red (#DC2626), soft gradients
- Glassmorphism effects with backdrop blur
- Framer Motion animations throughout
- Mobile-first responsive design
- Smooth scrolling with section-based storytelling

### Language Support
- **Primary:** French (default)
- **Secondary:** English
- Premium language switcher (EN | FR) in top-right
- Smooth transitions without page reload

### Key Differentiator
- **SenDon-AI** - Predictive intelligence for blood shortage forecasting
- Emphasis on proactive vs reactive healthcare
